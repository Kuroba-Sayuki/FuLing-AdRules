- **茯苓规则**为`项目维护者`网络冲浪时抓取。
    - 自`2025年04月28日`起，本仓库`不再提供带有条目注释`的规则列表。

- **非茯苓规则**为`下列情况其它作者规则`的收集。
    - 某些规则的旧版本。
    - 某些规则的合并版本。
    - 已经找不到出处的规则。
- **部分`非茯苓规则`存在原项目地址，推荐**`优先使用`**原项目。**
